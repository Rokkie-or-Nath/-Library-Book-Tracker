LIBRARY BOOK TRACKER
Library Book Tracker is a simple and intuitive application designed to help users manage library books efficiently. 
Built by the EduCore team, it allows tracking, searching, and organizing books in a user-friendly interface.

FEATURES
-Add Books – Add new books with details like title, author, and genre.
-Search Books – Quickly search for books using keywords.
-Remove Books – Delete books that are no longer available.
-Update Book Details – Edit book information when necessary.
-User-Friendly UI – Easy-to-navigate interface for all users.
